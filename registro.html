Términos y condiciones de uso

Sistema de Biblioteca Escolar – Sebastián Nelson Jesús Bertello

Al instalar y utilizar este programa, usted acepta las siguientes condiciones:

Uso autorizado
El software está diseñado exclusivamente para la gestión de bibliotecas escolares.
Solo podrá ser utilizado por instituciones o personas debidamente autorizadas.

Responsabilidad del usuario
El autor no se hace responsable por:
El lugar en el que se instale el programa.
El uso que se le dé al mismo.
Pérdida, alteración o divulgación de información administrada por la institución.
El correcto resguardo de datos (respaldos, contraseñas, permisos) depende únicamente de la institución usuaria.

Contribución obligatoria
El uso del software implica el compromiso de realizar una contribución mensual.
La contribución inicial será de 2000 (dos mil) pesos argentinos, monto que podrá actualizarse en el futuro.

Comunicación con el servidor
Cada copia del programa se conecta periódicamente a un servidor de control para:
Verificar actualizaciones, parches y mejoras.
Recibir instrucciones de mantenimiento o seguridad.
Esta función es obligatoria para garantizar el correcto funcionamiento del software.

Limitaciones
El software no podrá ser modificado, redistribuido ni utilizado con fines distintos a los establecidos.
Queda prohibido el acceso no autorizado a la consola de administración remota.

Privacidad y datos
La aplicación administra información de libros y usuarios bajo responsabilidad exclusiva de la institución.
No se transmiten datos personales a terceros sin autorización expresa.

Aceptación
Si usted continúa con la instalación y el uso del programa, se entiende que acepta íntegramente estas condiciones.
En caso contrario, deberá abstenerse de instalar o utilizar el sistema.
